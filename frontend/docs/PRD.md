# Orange 财务收款管理系统 - 产品需求文档 (PRD)

**版本**: v0.0.1  
**日期**: 2026-01-03  
**状态**: 原型完成

---

## 1. 产品概述

### 1.1 产品定位
Orange 是一款面向中小企业和自由职业者的**项目收款管理系统**，帮助用户追踪项目合同、管理收款计划、分析财务数据。

### 1.2 目标用户
| 用户类型 | 描述 |
|---------|------|
| 项目经理 | 需要追踪多项目收款进度 |
| 财务人员 | 需要管理应收账款和收款记录 |
| 自由职业者 | 需要简单的项目和收款管理工具 |
| 小微企业主 | 需要全面的收款概览和分析 |

### 1.3 核心价值
- **可视化收款进度** - 一目了然的项目收款状态
- **到期提醒** - 防止遗漏收款
- **数据分析** - 洞察收款趋势和问题

## 2. 功能需求

### 2.1 用户认证模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 用户登录 | 账号密码登录，支持记住我 | P0 |
| 登录状态保持 | 刷新页面保持登录状态 | P0 |
| 退出登录 | 清除登录态并跳转登录页 | P0 |

### 2.2 工作台模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 统计卡片 | 展示合同总额、已收款、待收款、收款率 | P0 |
| 收入趋势图 | 图表展示近期收款趋势 | P1 |
| 近期项目 | 列表展示最近更新的项目 | P1 |
| 即将到期收款 | 列表展示即将到期的收款计划 | P0 |
| 快捷操作 | 新建项目、添加收款的快捷入口 | P1 |

### 2.3 项目管理模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 项目列表 | 展示所有项目，支持筛选 | P0 |
| 项目分页 | 每页10条，支持翻页 | P1 |
| 新建项目 | 填写项目信息创建新项目 | P0 |
| 编辑项目 | 修改已有项目信息 | P0 |
| 删除项目 | 删除项目（需确认） | P1 |
| 项目详情 | 查看项目完整信息和收款记录 | P0 |
| 收款进度 | 可视化展示项目收款完成度 | P1 |

### 2.4 收款管理模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 添加收款记录 | 填写收款信息添加到项目 | P0 |
| 收款列表 | 在项目详情中展示收款记录 | P0 |
| 收款状态 | 标记已收款/待收款状态 | P0 |

### 2.5 收款日历模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 日历视图 | 按月展示收款计划 | P1 |
| 日期标记 | 标记有收款的日期 | P1 |
| 今日指示 | 突出显示当天 | P2 |

### 2.6 数据分析模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 收款趋势图 | 展示历史收款趋势 | P1 |
| 收款率分析 | 分析收款完成率 | P1 |
| 项目分布 | 分析项目状态分布 | P2 |

### 2.7 系统设置模块

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 个人信息 | 查看和修改个人信息 | P1 |
| 字典管理 | 管理系统下拉选项 | P2 |
| 外观设置 | 主题切换（明/暗） | P2 |

## 3. 非功能需求

### 3.1 性能要求
- 页面首次加载 < 3秒
- 页面切换动画流畅 (60fps)

### 3.2 兼容性要求
- 支持 Chrome、Safari、Firefox、Edge 最新版本
- 响应式布局适配 375px - 1920px+ 屏幕

### 3.3 用户体验要求
- 采用 Liquid Glass 设计语言
- 支持深色/浅色主题
- 所有操作需有明确反馈

## 4. 数据字典

### 4.1 项目状态
| 值 | 显示名称 | 说明 |
|----|---------|------|
| notstarted | 未开始 | 项目尚未启动 |
| active | 进行中 | 项目正在执行 |
| completed | 已完成 | 项目已交付 |
| pending | 即将交付 | 项目接近完成 |

### 4.2 收款阶段
| 值 | 显示名称 |
|----|---------|
| deposit | 首付款 |
| progress | 进度款 |
| final | 尾款 |
| warranty | 质保金 |

### 4.3 支付方式
| 值 | 显示名称 |
|----|---------|
| bank_transfer | 银行转账 |
| alipay | 支付宝 |
| wechat | 微信支付 |
